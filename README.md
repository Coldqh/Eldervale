# Eldervale

Eldervale — браузерная процедурная симуляция живого фэнтези-мира. Она создаёт именных жителей, династии, государства, поселения, армии, войны, чудовищ, драконов, артефакты, книги, подземелья, торговые пути и причинно связанную хронику.

## Локальный запуск

```bash
npm ci
npm run dev
```

## Сборка

```bash
npm run build
```

Проект настроен для GitHub Pages по адресу `/Eldervale/`. В комплект входят PWA-манифест, Service Worker, обязательная проверка версии, автоматическая очистка старого кэша и адаптивный интерфейс под iPhone 14 Pro, другие телефоны и ПК.

## Версия 0.3.0 — «Живая история мира»

- детерминированная генерация по ключу мира;
- карта земли, владений, населения, опасности и торговли;
- каждый житель имеет имя, семью, отношения, профессию, богатство, верность, травмы и биографию;
- династии, наследники, притязания и смена правящего дома;
- ресурсы поселений, запасы, голод, беспорядки и торговые пути;
- войны с причиной, целью, спорной территорией, армиями, снабжением, сражениями и мирными условиями;
- драконы и чудовища с территорией, голодом, поведением, целью и логовом;
- отправка героев на охоту и постоянные последствия победы, смерти или ранения;
- книги, которые ссылаются на реальные события и могут быть предвзяты;
- артефакты с цепочкой владельцев;
- появление руин из погибших поселений;
- месячная, годовая и десятилетняя симуляция в Web Worker;
- сохранение в IndexedDB, экспорт и импорт мира;
- обязательные обновления приложения и отображение текущей версии в настройках.

## Версия 0.3.2
