Eldervale 1.1.4 — стабилизация генерации

Исправляет:
- ложные 100% до завершения индексации, передачи и сохранения;
- пик памяти из-за второй полной копии мира в Worker;
- автообновление во время генерации/загрузки/сохранения;
- тяжёлый Blob для каждой сущности при сохранении;
- временный массив getAll() при загрузке IndexedDB;
- один маркер с 80 травами: ресурсы распределяются пятнами по многим клеткам.

Применение из C:\Eldervale:
  powershell -ExecutionPolicy Bypass -File .\apply_patch.ps1
  npm ci
  npm run build
